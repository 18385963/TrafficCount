To use the automatic update, you need to load 'TrafficUpdate-alpha.lsp' to you AutoCAD application

steps:
1. Open AutoCAD
2. Type APPLOAD in the command line
3. Click 'Startup Suite' and 'Add' 
4. Navigate to the AutoLisp folder (This directory)
5. Add TrafficUpdate-alpah.lsp to startup suite
6. Restart AutoCAD
